棉花Bot介绍

这是一个什么样的Bot？​

棉花Bot 是一款基于 QQ 客户端的机器人，通过后端（也就是第三方工具）监听并接收消息，交由处理框架以实现各类功能的QQbot。

此Bot采用QQ个人号部署，在保障核心功能的同时，尽可能减少使用限制。

为啥是个人号？因为QQ官方机器人的限制太大了，先不说各种限制，刚接个 AI 就可能被封，咱确实有点玩不起 🥲

棉花Bot运行在服务器（通俗点来讲就是电脑）上，服务器通过24h不间断开机来维持服务。

只要服务器一直开着，机器人就能一直响应你的请求。

此Bot的部署方向​

目前，本 Bot 主要定位为一个指令型助手，因此在 AI 能力方面暂时不会投入太多精力去深度优化。

不过本Bot的指令丰富较为完善，可以满足大部分人的需求，包括B站视频识别，抖音视频识别，音乐下载，涩图获取等一系列指令

未来，我会继续以新增有趣、实用的功能为主

当然，如果大家对 AI 交互体验有期待，我们也会认真考虑，在合适的时候慢慢优化

我该如何使用​

  1. 输入 help 查看使用说明，输入 //menu 查看可以使用的指令
  2. 可以查看本文档的指令使用示例（本文档还在编写中）

有人想问：这么好的机器人，它收费吗？​

作为部署者，我的回答是：不可能，以后也绝不收费。

我见过部分Bot，只有收费才可以使用，把真正想要体验的用户挡在了外面。我想通过部署这样一个Bot，让更多人能够免费用上。

我知道有些收费项目可能会对免费Bot有看法，所以我在这里声明：本Bot所使用的框架（以及插件）和后端，完全是开源项目！源代码可以在GitHub上看到。

当然，免费并不意味着你可以把“免费”当作理所当然，可以随意滥用。
在使用Bot时，请遵守 help 和 //邀群须知 中的内容，否则可能会被暂时或长期限制使用

💡 输入 //邀群须知 可以查看具体内容。
